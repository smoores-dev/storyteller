# Request a Feature

Feature requests help us understand what matters most to the community. We use
[GitLab Issues](https://gitlab.com/storyteller-platform/storyteller/-/issues)
and the `#feature-requests` channel on
[Discord](https://discord.com/invite/KhSvFqcrza) to track them.

:::info[Search first before opening a new request]

[Search existing issues](https://gitlab.com/storyteller-platform/storyteller/-/issues)
and check `#feature-requests` on
[Discord](https://discord.com/invite/KhSvFqcrza). If you find a match, give it a
thumbs up 👍 or add your thoughts to show interest.

:::

## Create a New Issue

If your feature isn't already requested, open a new issue with:

- A clear description of what you'd like added or changed.
- Why this feature would be valuable.
- Examples or mockups if possible.
