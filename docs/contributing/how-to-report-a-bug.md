# Report a Bug

Reporting bugs helps improve Storyteller. We use
[GitLab Issues](https://gitlab.com/storyteller-platform/storyteller/-/issues) to
track them.

:::info[Search first before opening a new issue]

[Search existing issues](https://gitlab.com/storyteller-platform/storyteller/-/issues).
If you find a match, give it a thumbs up 👍 or add details to show it affects
multiple people.

:::

## Create a New Issue

If your bug isn't already reported, open a new issue with:

- A clear description of the problem.
- Steps to reproduce the bug.
- Screenshots, videos, or server logs if applicable.
